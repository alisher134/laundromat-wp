# About Items Icon Guide

Now that About Items use image uploads instead of SVG code, here's how to prepare and upload icon images.

## üìê Icon Specifications

### Recommended Format
- **Format**: PNG with transparent background
- **Size**: 100x100px (minimum), or any square size up to 500x500px
- **File size**: Under 50KB
- **Color**: Blue (#3A6D90) or any color that matches your brand
- **Background**: Transparent

### Why These Specs?
- PNG supports transparency (looks clean on any background)
- Square ratio ensures consistent sizing
- Small file size = fast page loading
- Brand colors maintain visual consistency

## üé® How to Get Icons

### Option 1: Download Free Icons (RECOMMENDED)

#### Flaticon (Best for styled icons)
1. Visit https://www.flaticon.com/
2. Search for your icon (e.g., "clock", "timer", "price tag")
3. Select an icon you like
4. Click "Download PNG"
5. Choose size: 128px or 256px
6. Download and save

**Suggested icons for existing cards:**
- "365 days" ‚Üí Search "24-7" or "always open" or "clock"
- ">60 min" ‚Üí Search "timer" or "stopwatch" or "fast"
- "low prices" ‚Üí Search "price tag" or "discount" or "coin"

#### Icons8 (Great variety)
1. Visit https://icons8.com/icons
2. Search for your icon
3. Select icon style (we recommend "iOS" or "Material Rounded")
4. Click icon ‚Üí Download
5. Choose PNG format
6. Select size 100px or larger
7. Download

#### The Noun Project (Simple, clean icons)
1. Visit https://thenounproject.com/
2. Search for icon
3. Free accounts can download PNGs
4. Choose "Download as PNG"

### Option 2: Convert Existing SVG to PNG

If you already have SVG icons:

1. Visit https://cloudconvert.com/svg-to-png
2. Upload your SVG file
3. Set width/height to 100px
4. Convert and download PNG

### Option 3: Create Custom Icons

#### Using Canva (No design skills needed)
1. Go to https://www.canva.com/
2. Create new design ‚Üí Custom size ‚Üí 100x100px
3. Search "elements" for icons
4. Add icon to canvas
5. Resize to fill canvas
6. Change color to #3A6D90
7. Download as PNG (with transparent background)

#### Using Figma (For designers)
1. Create 100x100px artboard
2. Design or import icon
3. Export as PNG (2x or 3x for retina)
4. Ensure background is transparent

## üì§ How to Upload Icons to WordPress

### Step 1: Prepare Your Icon Files
1. Rename files descriptively:
   - `icon-247-open.png`
   - `icon-fast-service.png`
   - `icon-low-prices.png`

### Step 2: Upload to WordPress
1. Go to **WordPress Admin ‚Üí Media ‚Üí Add New**
2. Drag and drop your PNG files
3. Wait for upload to complete
4. Click on each uploaded image to verify

### Step 3: Assign to About Items
1. Go to **About Items**
2. Click on an item to edit
3. Look for **"Featured Image"** box in right sidebar
4. Click **"Set featured image"**
5. Select your uploaded icon
6. Click **"Set featured image"** button
7. Click **"Update"** to save

**That's it!** The icon will now appear on your homepage.

## üé® Making Icons Match Your Brand

If you have PNG icons but want to change the color:

1. Visit https://www.aconvert.com/image/png-editor/
2. Upload your PNG
3. Use "Fill" tool
4. Select brand color (#3A6D90)
5. Fill the icon shape
6. Download

### Using Photoshop (Advanced)
1. Open PNG in Photoshop
2. Select ‚Üí Color Range ‚Üí Click icon color
3. Create new layer
4. Fill selection with #3A6D90
5. Export as PNG

### Using Free Online Editor
1. Visit https://www.photopea.com/ (free Photoshop alternative)
2. Open your PNG
3. Use Magic Wand tool to select icon
4. Fill with new color #3A6D90
5. Export as PNG

## üí° Icon Best Practices

### ‚úÖ DO:
- Use simple, clear icons (not complex illustrations)
- Keep icons consistent in style across all cards
- Use brand colors or neutral colors
- Test on both light and dark backgrounds
- Use transparent backgrounds

### ‚ùå DON'T:
- Use photos or complex images
- Mix different icon styles (outlined + filled)
- Use text in icons (use the title/description instead)
- Use very detailed icons (they'll be small)
- Forget to test how they look on the page

## üìè Testing Your Icons

Before finalizing, check:

1. **Size**: Do they look clear at small size?
2. **Color**: Do they contrast well with background?
3. **Style**: Do all three icons match stylistically?
4. **Spacing**: Do they have enough padding/breathing room?
5. **Mobile**: Check how they look on phone screens

### Quick Preview
Save your icons in the same folder and view them side by side. They should:
- Look like they belong to the same family
- Be equally detailed (or simplified)
- Use similar line weights
- Have similar color intensity

## üÜò Troubleshooting

**Icon not showing?**
- Make sure you clicked "Set featured image" (not just uploaded)
- Verify image uploaded successfully to Media Library
- Check that image URL works when clicked in Media Library

**Icon too small/large?**
- Minimum recommended: 100x100px
- Maximum recommended: 500x500px
- The CSS will automatically resize it properly

**Icon has white background?**
- PNG file doesn't have transparency
- Re-export with transparent background
- Or use image editor to remove background

**Icon wrong color?**
- Use one of the color-change methods above
- Or download a different version of the icon

**Icons don't match style?**
- Download all icons from the same icon set
- Use the same icon "family" (e.g., all Heroicons, all Material Icons)
